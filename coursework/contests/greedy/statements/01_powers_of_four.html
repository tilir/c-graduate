Problem EV — степени четвёрки

Напишите функцию, которая оптимально раскладывает число по ограниченному количеству степеней четвёрки

Например 27 = 1*16 + 2*4 + 3*1

Но также 27 = 1*16 + 1*4 + 7*1

Будем считать каждую степень четвёрки одной монетой. Тогда в первом разложении использовано 6 монет, а во втором 9 монет. Функция должна находить минимальное количество монет для заданного числа (n) и максимальной степени (m)

unsigned powers_of_four(unsigned n, unsigned m)

Посылка должна состоять из этой одной функции (допустимо также любое количество вспомогательных), и не должна содержать функции main


Пример 1 (13 == 3*4 + 1)

Вход: n = 13, m = 1

Выход: 4


Пример 2 (27 == 6*4 + 3*1)

Вход: n = 27, m = 1

Выход: 9


Пример 3 (27 == 1*16 + 2*4 + 3*1)

Вход: n = 27, m = 2

Выход: 6
